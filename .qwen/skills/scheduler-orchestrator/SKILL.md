# Scheduler Orchestrator

Schedule automated tasks for the AI Employee.

## Overview

This skill enables Qwen Code to:
- Schedule recurring tasks (hourly, daily, weekly)
- Trigger watcher scripts on schedule
- Generate periodic reports
- Run maintenance tasks
- Orchestrate multi-step workflows

## Scheduled Tasks

| Task | Frequency | Description |
|------|-----------|-------------|
| **Dashboard Update** | Hourly | Refresh stats and counts |
| **Process Actions** | Every 30 min | Process /Needs_Action files |
| **Email Check** | Every 15 min | Check for new emails |
| **Daily Briefing** | Daily 8 AM | Generate morning report |
| **Weekly Audit** | Weekly Monday | Business and accounting audit |
| **Cleanup** | Weekly Sunday | Archive old files |

## Usage

### Schedule a Task (Windows)

```bash
# Create scheduled task for dashboard update
schtasks /create /tn "AI_Employee_Dashboard_Update" /tr "python watchers\action_processor.py" /sc hourly /ru SYSTEM
```

### Schedule a Task (Linux/Mac)

```bash
# Add to crontab
crontab -e

# Dashboard update every hour
0 * * * * cd /path/to/project && python watchers/action_processor.py

# Daily briefing at 8 AM
0 8 * * * cd /path/to/project && python watchers/generate_briefing.py

# Weekly audit on Monday 9 AM
0 9 * * 1 cd /path/to/project && python watchers/weekly_audit.py
```

## Task Definitions

### Hourly Dashboard Update

```python
# schedulers/hourly_dashboard.py
#!/usr/bin/env python3
"""Update Dashboard.md every hour."""

import sys
from pathlib import Path
from datetime import datetime

def update_dashboard():
    vault_path = Path(sys.argv[1]) if len(sys.argv) > 1 else Path("AI_Employee_Vault")
    dashboard = vault_path / "Dashboard.md"
    
    # Count files
    needs_action = len(list((vault_path / "Needs_Action").glob("*.md")))
    done_today = len([f for f in (vault_path / "Done").glob("*_done_*.md")
                      if datetime.now().strftime("%Y%m%d") in f.name])
    pending = len(list((vault_path / "Pending_Approval").glob("*.md")))
    
    # Update dashboard
    content = dashboard.read_text(encoding="utf-8")
    # ... update stats ...
    dashboard.write_text(content, encoding="utf-8")
    
    print(f"Dashboard updated: {needs_action} pending, {done_today} done")

if __name__ == "__main__":
    update_dashboard()
```

### Weekly Audit Script

```python
# schedulers/weekly_audit.py
#!/usr/bin/env python3
"""Generate weekly business audit every Monday."""

import sys
from pathlib import Path
from datetime import datetime, timedelta

def weekly_audit():
    vault_path = Path(sys.argv[1]) if len(sys.argv) > 1 else Path("AI_Employee_Vault")
    briefings = vault_path / "Briefings"
    briefings.mkdir(exist_ok=True)
    
    today = datetime.now()
    week_ago = today - timedelta(days=7)
    
    audit_file = briefings / f"{today.strftime('%Y-%m-%d')}_Weekly_Audit.md"
    
    # Count tasks completed this week
    done_folder = vault_path / "Done"
    completed = [f for f in done_folder.glob("*_done_*.md")
                 if week_ago.strftime("%Y%m%d") in f.name or
                    today.strftime("%Y%m%d") in f.name]
    
    content = f"""---
type: weekly_audit
week_start: {week_ago.strftime('%Y-%m-%d')}
week_end: {today.strftime('%Y-%m-%d')}
generated: {today.isoformat()}
---

# 📊 Weekly Business Audit

**Period:** {week_ago.strftime('%b %d')} - {today.strftime('%b %d, %Y')}

---

## 📈 Key Metrics

| Metric | This Week | Last Week | Change |
|--------|-----------|-----------|--------|
| Tasks Completed | {len(completed)} | - | - |
| Revenue | $X,XXX | $X,XXX | +X% |
| Response Time | X hours | X hours | -X% |

---

## ✅ Completed Tasks

<!-- List top accomplishments -->

---

## 🎯 Goals Progress

<!-- Update goal tracking -->

---

## 🚨 Bottlenecks Identified

<!-- List any delays or issues -->

---

## 💡 Recommendations

<!-- AI suggestions for improvement -->

---

*Generated by AI Employee Weekly Audit*
"""
    
    audit_file.write_text(content, encoding="utf-8")
    print(f"Weekly audit generated: {audit_file}")

if __name__ == "__main__":
    weekly_audit()
```

## Windows Task Scheduler

### Create Tasks via PowerShell

```powershell
# Hourly Dashboard Update
$action = New-ScheduledTaskAction -Execute "python" `
    -Argument "watchers\action_processor.py" `
    -WorkingDirectory "E:\GIAIC\Quarter-04\ai_employee_fte_Hackathon_0"
$trigger = New-ScheduledTaskTrigger -Hourly -At 0
Register-ScheduledTask -TaskName "AI_Employee_Hourly" `
    -Action $action -Trigger $trigger -RunLevel Highest

# Daily Briefing at 8 AM
$action = New-ScheduledTaskAction -Execute "python" `
    -Argument "watchers\daily_briefing.py" `
    -WorkingDirectory "E:\GIAIC\Quarter-04\ai_employee_fte_Hackathon_0"
$trigger = New-ScheduledTaskTrigger -Daily -At 8am
Register-ScheduledTask -TaskName "AI_Employee_Daily" `
    -Action $action -Trigger $trigger -RunLevel Highest

# Weekly Audit on Monday 9 AM
$action = New-ScheduledTaskAction -Execute "python" `
    -Argument "watchers\weekly_audit.py" `
    -WorkingDirectory "E:\GIAIC\Quarter-04\ai_employee_fte_Hackathon_0"
$trigger = New-ScheduledTaskTrigger -Weekly -DaysOfWeek Monday -At 9am
Register-ScheduledTask -TaskName "AI_Employee_Weekly" `
    -Action $action -Trigger $trigger -RunLevel Highest
```

## Error Handling

- **Task fails:** Log error, retry once, notify if still failing
- **Missed schedule:** Run immediately, adjust next schedule
- **Resource locked:** Wait 5 minutes, retry
- **Script error:** Log full traceback, create error file

## Best Practices

1. **Log everything** - Capture stdout/stderr to log files
2. **Handle errors** - Don't let one failure stop all tasks
3. **Monitor status** - Check task execution regularly
4. **Rotate logs** - Clean up old log files
5. **Test schedules** - Verify tasks run as expected
